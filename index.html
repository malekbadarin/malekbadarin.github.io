HTML

Here is a list of the most used HTML tags

HTML table tags

HTML tag Function
<table> Initiates a table
<thead> Contains the table's header
<tbody> Contains the table's main data
<th> Defines column headers
<tr> Creates a new row
<td> Creates a new field in the row

Semantic HTML tags

These are HTML tags that are used instead of <div> tags. They offer an alternative that is more readable.

HTML tag Use case Example
<header> Used for the header section of the page <header>
<h1>This is the Title of the page</h1>
<h3>This is the sub-heading of the page.</h3>
</header>
<nav> Used for the navigation section of the page <nav>
<ul>
<li>About page</li>
<li>Contace page.</li>
</ul>
</nav>
<article> Used for a self contained/independant section <article>
<h3>This is the heading of the article</h3>
<p>This is the meat of the article.</p>
</article>
<aside> Used for a remark that is tangentially relevant, but not completely indepnedant of the context <p>This is a paragraph of text.</p>
<aside>This is a remark that is relevant to the paragraph above.</aside>
<footer> Used for the footer section of the page <footer>This is the footer of the page, could contain contact details and developer information.</footer>

CSS

Note that the general structure of a CSS rule is as follows:

selector {
property: value;
}

Here is a list of the most used CSS selectors and selection methods:

Selector Function example
tag Selects all instances of this tag a {
color: blue;
}
.class Selects all elements of that specific class .red-buttons {
color: blue;
}
#id Selects the element with that specific id #html-tbl {
color: blue;
}
parent child Selects all instance of the child selector inside the parent selector header a {
color: blue;
}
first.second Selects all elements that are simultaneously the first and second selectors .external-links.a {
color: blue;
}
first, second Selects all elements that are either the first of the second a, p {
color: blue;
}
* Selects all elements (can be combined with other selctors) * {
color: blue;
}

footer * {
color: blue;
} (This selects all elements inside "footer")
selector:pseudo-class Selects all elements that are modified by the pseudo-class a:hover {
color: blue;
}
Property Function Acceptable values Example How it is rendered
color Defines the foreground color of the element Takes named, RGB, RGBA, HEX, HSL, or HSLA color values color: red; This text is red
background-color Defines the background color of the element Takes named, RGB, RGBA, HEX, HSL, or HSLA color values background-color: red; The background here is red
border Defines the border of the element. It can set the thickness, style, and color of the border Thickness: takes measurments in px
Style: solid, dotted, dashed, grooved, double, ridge, inset, outset, hidden, none
Color: takes named, RGB, RGBA, HEX, HSL, or HSLA color values		 border: 5px dotted red; This felid has a 5px-thick, dotted, red border
background-image Defines the background image of the element Takes a URL of the image location using url() background-image: url(./resources/cat.png) This field has a cat background
font-family Defines the font-family of the element Takes named font-family, serif, or sans-serif. Values can be stacked with a "," for fallback support font-family: "Comic Sans MS", sans-serif; The font family of this field is "Comic Sans MS", if it is not available, it will display the default sans-serif font.
font-size Defines the size of the font of the element Takes measurments in px (this is the default rem measurment of the element) font-size: 12px; The font size of this field is 12px
